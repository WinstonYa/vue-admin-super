## Getting started
```bash
# clone the project
git clone git@github.com:destiny-miracle/vue-admin-super.git

# enter the project directory
cd vue-admin-super

# install dependency
npm install

# develop
npm run serve
```

## Build
```bash
npm run build
```

## Advanced
```bash
# code format check
npm run lint

# code format check and auto fix
npm run lint --fix
```
